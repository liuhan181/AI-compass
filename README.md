# AI Compass


🧠 **产品介绍**  
**AI Compass** 是一款面向 AI Chat 场景的本地上下文管理插件，不仅仅是摘录工具，而是辅助用户管理 AI 对话过程的「第二大脑」。帮助用户在长对话中收集、分类、导出关键内容，降低信息过载带来的上下文负担。


⚙️ **核心功能**
- ✅ 平台适配：支持ChatGPT、豆包等 AI 对话页面
- ✅ 内容管理：选区收藏、富文本保留、6类内容分类、相似内容智能检测
- ✅ 体验优化：会话隔离、页面刷新不丢失数据
- ✅ 导出能力：本地存储、一键 Markdown 复制 & 导出
- 🧪 测试功能：原文定位（Beta 增强能力）


🔒 **隐私说明**
数据默认本地存储，全程不上传云端，保护你的对话内容安全。
